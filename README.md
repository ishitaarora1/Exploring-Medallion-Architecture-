# Exploring-Medallion-Architecture-
A hands-on Azure Data Engineering project implementing the Medallion Architecture (Bronze–Silver–Gold) using Azure Data Factory, Databricks, and ADLS , from raw GitHub data ingestion to transformation and visualization.

# Contents:

## Dataset 

Access the public dataset under the guidance of Ansh Lamba: [Adventure‑Works Data Engineering Project](https://github.com/anshlambaoldgit/Adventure-Works-Data-Engineering-Project/tree/main/Data)


## Silver Notebook (.html):

Contains the cleaned and transformed data at the Silver layer of the Medallion architecture.

Can be accessed directly in Databricks by importing/opening the HTML notebook.

## Pipeline (.json):

Contains the exported pipeline configuration for data ingestion/processing.

Can be imported into Azure Data Factory (ADF) to replicate the pipeline workflow.

No sensitive keys or secrets are included in the exported JSON.
